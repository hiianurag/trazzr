# Trazzr - Single Page Application

Trazzr is a modern, responsive Single Page Application (SPA) for a premium Ayurvedic herbal products company. This project is built using pure HTML, CSS, and Vanilla JavaScript, ensuring high performance and a clean codebase without the overhead of heavy frameworks.
